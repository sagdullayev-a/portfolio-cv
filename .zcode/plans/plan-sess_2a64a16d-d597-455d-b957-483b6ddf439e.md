# Telegram Contact Form Integratsiyasi Rejasi

## Maqsad
Contact formadagi xabarlar (name + email + message) sizning Telegram shaxsiy chattingizga tushsin. WhatsApp butunlay olib tashlanadi.

## Texnik yondashuv
- **Vercel Serverless Function** (`api/send-telegram.ts`) — bot tokenni serverda saqlaydi (xavfsiz, frontend'da ko'rinmaydi)
- Frontend formadan `POST /api/send-telegram` so'rov yuboradi
- Bot sizning Telegram chat_id'ingizga xabar yuboradi

---

## 1-fayl: `api/send-telegram.ts` (YANGI)
Vercel serverless function:
- POST body: `{ name, email, message }`
- Validatsiya: 3 maydon ham bo'sh emas, email formati to'g'ri
- `process.env.TELEGRAM_BOT_TOKEN` va `process.env.TELEGRAM_CHAT_ID` ni o'qiydi
- `https://api.telegram.org/bot<token>/sendMessage` ga fetch qiladi
- Xabar formati:
  ```
  📨 Yangi xabar — Portfolio

  👤 Ism: Aziz
  ✉️ Email: aziz@mail.com

  💬 Xabar:
  Salam! Loyiha bormi?
  ```
- Javob: `{ ok: true }` yoki `{ ok: false, error: "..." }`
- `vercel.json`'ga `api/send-telegram` route'ini qo'shmaymiz — Vercel avtomatik `api/` papkani serverless function deb tanidi (SPA rewrite esa `/api` bilan boshlanmaydiganlar uchun ishlayveradi).

## 2-fayl: `src/components/ContactSection.tsx` (O'ZGARTIRISH)
- `form` state'ga `email` qo'shamiz: `{ name, email, message }`
- `handleSend` ni `async` qilib, `/api/send-telegram`'ga fetch:
  - try/catch
  - `status` state: `"idle" | "sending" | "success" | "error"`
  - Sending paytida button disabled + spinner matn ("Sending...")
  - Success: forma tozalanadi, "Xabar yuborildi ✅" toast/matn ko'rsatadi (3 soniyadan keyin reset)
  - Error: "Xatolik yuz berdi, qayta urinib ko'ring" matn
- Email input maydoni qo'shamiz (glass card style'da, name bilan bir xil)
- Email majburiy emas — lekin agar to'ldirilgan bo'lsa, formati valid bo'lishi kerak. Aslida emailni **majburiy** qilamiz, chunki javob berishingiz kerak. (Agar majburiy emas xohlasangiz ayting.)
- WhatsApp deep-link (`window.open wa.me`) butunlay olib tashlanadi
- `FaWhatsapp` import olib tashlanadi
- Mavjud UI dizayni (glass card, glossy button, framer-motion) saqlanadi

## Siz qilishingiz kerak bo'lgan ishlar (koddan tashqari)

### 1. Telegram Bot yaratish
1. Telegram'da **@BotFather**'ni oching
2. `/newhook` emas — `/newbot` ni yuboring
3. Bot nomi (display): `Azizxon Portfolio Bot` (xohlaganingiz)
4. Bot username: `azizxon_portfolio_bot` (tugashi `_bot` bo'lishi shart, noyob bo'lishi kerak)
5. BotFather **token** beradi: `8123456789:AAH...` — saqlang

### 2. O'z chat_id'ingizni olish
1. O'zingiz yaratgan botga **bir marta** xabar yozing (`/start` yoki istalgan matn) — bu majburiy, aks holda bot sizga yozolmaydi
2. Brauzerda oching:
   `https://api.telegram.org/bot<SIZNING_TOKEN>/getUpdates`
3. Javobda `"chat":{"id": 123456789, ...}` ko'rasiz — o'sha `id` sizning `chat_id`'ingiz

### 3. Vercel'ga env o'zgaruvchilarni qo'shish
- Vercel Dashboard → sizning project → **Settings → Environment Variables**
- `TELEGRAM_BOT_TOKEN` = sizning token
- `TELEGRAM_CHAT_ID` = sizning chat_id
- Keyin **Redeploy** qiling (env o'zgaruvchilar deploy vaqtida o'qiladi)
