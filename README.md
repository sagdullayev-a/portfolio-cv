## 🛠️ Loyihada Ishlatilgan Texnologiyalar (Tech Stack)

Ushbu portfolio veb-sayti eng zamonaviy texnologiyalar va kutubxonalar yordamida, yuqori sifatli animatsiya hamda 3D interaktivlik asosida yaratilgan:

### ⚡ Core & Build
*   **React 19:** Foydalanuvchi interfeysini (UI) boshqarish.
*   **Vite:** Juda tez ishlovchi loyiha yig'uvchi (build tool).
*   **TypeScript:** Statik tiplash va xavfsiz kod yozish tizimi.

### 🎨 Styling & Layout
*   **Tailwind CSS v4:** Tezkor, moslashuvchan (responsive) va zamonaviy glassmorphism (shisha) stillari.
*   **tw-animate-css:** Qo'shimcha tayyor CSS animatsiyalar.

### 📦 3D & Physics (Interaktiv ID Karta)
*   **Three.js & @react-three/fiber:** Brauzerda yuqori sifatli 3D grafikani render qilish.
*   **@react-three/drei:** 3D model, teksturalar va yorug'lik effektlarini boshqarish.
*   **@react-three/rapier:** ID kartani sichqoncha yordamida sudrab tebratish uchun 3D fizika dvigateli.
*   **meshline:** Karta bog'langan tasmani chizish.

### ✨ Animations (Harakatlar)
*   **Framer Motion:** Sichqoncha harakatiga mos parallax effektlar va sahifa bo'limlarining animatsion yuklanishi.
*   **GSAP (ScrollToPlugin):** Sahifa bo'limlariga silliq scroll qilish va maxsus effektlar.

### 🌍 i18n (Ko'p Tillilik)
*   **i18next / react-i18next:** Saytni 3 tilda (O'zbekcha 🇺🇿, Inglizcha 🇬🇧, Ruscha 🇷🇺) to'liq ishlatish va brauzer tilini avtomatik aniqlash.

### 🚥 Routing & Icons
*   **React Router DOM v7:** Sahifalararo navigatsiya va yo'nalishlarni boshqarish.
*   **Lucide React & React Icons:** Zamonaviy SVG ikonlar to'plami.
